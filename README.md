# Brazilian E-Commerce Analysis (Olist)

## Project Overview
This project analyzes Brazilian e-commerce data from Olist
to explore customer behavior, order characteristics,
and customer satisfaction using statistical methods.

## Dataset
Source: Kaggle – Brazilian E-Commerce Public Dataset (2016–2018)

## Planned Analyses
- Descriptive statistics
- Correlation analysis
- A/B testing
- ANOVA tests

## Tools
- Python
- Pandas, NumPy
- SciPy, Statsmodels
- Plotly
